# Saathvik Varma — Portfolio

A minimal, dark-themed personal portfolio website.

## 🚀 Deploy to GitHub Pages (Free)

### Step 1 — Create a GitHub repository
1. Go to [github.com](https://github.com) and sign in
2. Click **New repository**
3. Name it exactly: `saathvikvarma.github.io`
   *(must match your GitHub username for the free domain to work)*
4. Set it to **Public**
5. Click **Create repository**

### Step 2 — Upload the files
Upload all three files into the repository:
- `index.html`
- `style.css`
- `script.js`

You can drag-and-drop them directly on the GitHub website.

### Step 3 — Enable GitHub Pages
1. Go to your repository **Settings**
2. Scroll to **Pages** in the left sidebar
3. Under **Branch**, select `main` → `/ (root)` → click **Save**

### Step 4 — Visit your live site
After ~1 minute, your site will be live at:
`https://saathvikvarma.github.io`

---

## 📸 Adding Your Photo
1. Add your photo file to the repository (name it `photo.jpg`)
2. The site will automatically display it in the About section

## ✏️ Customising Content
All content is in `index.html` — edit any text directly.
Styles are in `style.css` — colors are controlled by CSS variables at the top.
